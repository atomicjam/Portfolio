May she grow in heavenly light

Co-Brand microsite for The Cheltenham Ladies' College

Brief

CLC Flourish is a long-term fundraising plan that responds to this vision and identifies how College might strengthen it's position with the help of our friends and supporters.

A micro site that can be managed by a separate 'Development' team with its own unique identity, yet familiar feel to the main College site.

Integrations should allow for the site to take gifts and donations using College current fundraising software Blackbaud

HTML Email templates flowing the brand guidelines for use in Blackbaud's NXT Marketing portal.

Stack

ginx Silverstripe
Github Docker

Extending

Created in 2016, development started life from a foundation of Silverstripe CMS using the College's main site as a core template, further details on how the core template was created are available on the Core Project page.

Aside from the visual updates for the Flourish brand, a number of project specific integrations were added to the core template.

Embedded .NET Framework components provided by Blackbaud, generate payment forms, additional CSS was added to overwrite the Blackbaud styles and make the forms a more seamless integration.

HTML Email templates matching the College's Mailchimp templates were created, with additional tooling developed to ease inlining styles to ensure the templates correctly rendered when sent using Blackbaud NXT.

Traditional printed materials were embedded into the site with the addition of a Yumpu template that allowed page turning experience for PDF files in the browser.
