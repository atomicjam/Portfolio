Organisation of Wellbeing & Enrichment activities for the school term requires large amounts of administration to ensure each activity is staffed, located, provisioned correctly so that events can run smoothly.

Brief

The site defines the view of modern education through sophisticated photography as a glimpse into life at College, aiming to dismiss may of the pre conceptions of what a boarding school is.

Development must be flexible enough to act as a versatile backdrop for the variety of co-brands and internal services.

Targeted at public presence, marketing & communication, brand should reflect an identity that feels elevated across both printed and digital brand touchpoints balancing the traditional and minimal.

Infrastructure

Created on a traditional LAMP stack in 2016, development started life from a foundation of Silverstripe CMS with custom page templates.

Initially locally hosted at the College data centre as a VMWare Instance. A modernisation plan in 2019 migrated the project to an EC2 AWS Instance for improved resilience and scale.

AWS additions included edge caching to improve the experience for overseas users with AWS Cloudfront, alongside security enhancements from AWS WAF & Shield.

To further reduce database latency and take advantage of AWS managed services MariaDB was swapped out for AWS RDS Database in 2020.

CI/CD was handled by Github and Github actions running a test suite developed in PHPUnit. Development was processed locally using Docker instances that match the AWS infrastructure, once tested changes are merged into the Main live branch.

In 2023 during a visual refresh the CMS was replaced by Statamic CMS, allowing the removal of the database entirely. Statamic further fitted with the College's strategic development goals using the Laravel framework to standardise development at CLC.

Front-end

Silverstripe page templates allow the External Relations team the ability to create content without technical experience. Components were created using standard HTML, SCSS & Native Javascript with branding options so they could be reused across multiple co-brands for consistency.

To facilitate collection of user data, "User Defined Forms" module augmented Silverstripe core functionality. the College has specific requirements around form completion, contributions to the open source project were made to add required functionality. My additions to the project was archived by Github into the Arctic Code Vault in 2020.

Once migrated to Statamic CMS, the additional functionality was included out the box many of the components were ported to Statamic Sets.

Tailwind CSS was introduced to replace SCSS to standardise the CSS that was being written, speed up development and reduce the total CSS bundle size.

The package manager used until that point had been Webpack, which was also update to ViteJS.

Javascript integrations were created to add additional third party data dynamically on the site, using API provided by each integrator. Social posts with infinite scrolling, news aggregation and rss support using Flockler.
Custom filterable calendar to show events from the College managed information system iSams and Templates to pull staff vacancies from HR system using Vacancy filler API.

Progressive enhancements to the site have been added over the life of the project improving accessability support, service workers as well as new functionality to enhance user engagement with the addition of pop-out notices, and more complex animations with Greensock.
